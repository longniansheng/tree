# @sinoui/tree

> 树组件旨在展现完整的层级关系，并具有展开、收起、选择等交互功能。

## 依赖安装

```shell
yarn add @sinoui/tree
```

## 级联选择

通过给`Tree`组件一个`cascade`属性实现级联选择。

<Playground>
  {() => {
    const loadChildren = (treeNode) => {
      if (!treeNode) {
        return getTreeNodes(0, 0, 'node', false).map((node) => ({
          ...node,
          selectType: 'checkbox',
          selectable: node.id !== 'node_9',
        }));
      } else {
        const promise = new Promise((resolve) => {
          setTimeout(() => {
            resolve(
              getTreeNodes(
                treeNode.level + 1,
                0,
                treeNode.id,
                false,
              ).map((node) => ({ ...node, selectType: 'checkbox' })),
            );
          }, 500);
        });
        return promise;
      }
    };
    return <div style={{height: '300px', overflowY:'auto'}}><Tree loadChildren={loadChildren} multiple cascade draggable /></div>;
  }}
</Playground>
